
Hệ thống hiện tại đã được tách thành 3 file script riêng biệt để bạn dễ dàng quản lý trong Google Apps Script:

1. Main_gs.txt -> Main.gs
2. Logic_gs.txt -> Logic.gs
3. Utils_gs.txt -> Utils.gs

Vui lòng tạo 3 file tương ứng trong trình soạn thảo Google Apps Script và copy mã nguồn từ các file txt này vào.
